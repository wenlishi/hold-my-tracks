# 应用截图

此目录用于存放个人轨迹管理与分析系统的应用截图。

## 截图说明

请将以下截图文件放入此目录：

- `login.png` - 登录页面截图
- `map-collect.png` - 地图采集页面截图
- `track-management.png` - 轨迹管理页面截图
- `track-detail.png` - 轨迹详情页面截图
- `heatmap.png` - 热力图展示截图
- `user-profile.png` - 个人中心页面截图

## 截图要求

- 格式：PNG 或 JPG
- 分辨率：建议 1080x1920 或更高
- 文件大小：每个文件不超过 2MB
- 截图内容：清晰展示应用界面和功能

## 如何获取截图

1. 在 HBuilderX 中运行项目
2. 使用模拟器或真机调试
3. 在需要截图的页面使用截图工具
4. 将截图文件重命名为对应的文件名
5. 放入此目录
